## 1. 本プロジェクトの概要（ビジネスの結論）
本プロジェクトでは、株式会社ZOZOの公開IRデータを活用し、前年比（YoY）の取扱高成長のドライバー（原動力）をデータから特定しました。

### 📊 分析によるコアの発見（Key Insight）
*   **結論：** 取扱高は前年比で伸びているものの、その中身は「新規会員の獲得」ではなく、**「既存ユーザーの年間購入回数の増加（リピート率向上）」**が牽引していることが判明。
*   **課題：** 1回あたりの平均出荷単価は横ばい, または微減傾向にあり、まとめ買い（クロスセル）の促進に課題があります。
*   **ビジネス提案：** 今後の売上最大化のためには、商品推薦（レコメンド）アルゴリズムの最適化による「合わせ買い」の誘導、およびブランド側への「コーディネート提案機能」の強化を推奨します。
